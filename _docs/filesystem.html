def cpu_use_stats()

source

cpu_use_stats

 cpu_use_stats ()

Returns CPU usage stats as a named tuple consisting of: - totalsecs : Total CPU seconds the analysis is based on. - all : sum of: user+syst+iowait+softirq percent use. - user : user % - system : system % - iowait : iowait % - softirq : softirq % - header : A header string for str. - str : string variable with fully formatted values for each of above.

#show_doc(cpu_use_stats)

Measure the average execution.

cpu_stats = cpu_use_stats()
95.6 µs ± 3.35 µs per loop (mean ± std. dev. of 7 runs, 1,000 loops each)

Print the header and str elememts. These can generally be used as is.

cpu_stats.header,cpu_stats.str
